# Next.js Scaffold Reference (Virtual Fisher)

Last updated: March 15, 2026

This folder is a plain `create-next-app` scaffold snapshot.

It is kept as a lightweight reference project and is **not** the active platform app. The active app is the repository root (`/`).

## What Is Here

- Default Next.js app-router starter files
- Its own `package.json`, `tsconfig`, and lint config
- No project-specific gameplay integration in this scaffold

## Run This Scaffold (Optional)

```bash
cd virtual-fisher/nextjs-scaffold
npm install
npm run dev
```

Open `http://localhost:3000`.

## Why It Exists

Use this folder if you want a clean baseline to compare against custom platform code in the root app.
